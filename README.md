# first-gui-game
My first GUI game built using Python and PyQt5. 
This proect is a number guessing game where the player recieves real-time feedback and interacts through a custom-designed graphical intterface. 

# Features 
- Interactive GUI built with PyQt5
- Real-time feedback for each guess
- Tracks up to 3 guesses
- Random number generation
- Customizable interface colours and characters
- New Game reset functionality

  # Technologies Used
  - Python
  - PyQt5

  # How to Run
  1. Install PyQt5:
     pip install PyQt5
  2. Run the game:
     python GuessingGame.py
     
  # What I leant:
  Building this project helped me understand how GUI applications respond to user interactions through event-driven programming. I learned how to connect buttons to functions, manage game state using class variables and dynamically update interface elements like text, images, and colours. This project also improved my ability to break down a large problem into smaller logical components and strengthened my confidence in building interactive software from scratch.
